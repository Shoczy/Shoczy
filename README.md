### 🔦 LumenAuth

A KeyAuth-style licensing & auth platform for developers who ship desktop
software (tools, loaders, commercial apps). Bind your app in a few lines and get
license keys, HWID locking, user logins, bans, gated downloads, reseller tokens,
webhooks and event logs — all from one liquid-glass dashboard.

- 🧩 **Official SDKs** → [`lumenauth-sdks`](https://github.com/Shoczy/lumenauth-sdks) · JS/TS · Python · Rust · C#
- 🌐 Language-agnostic HTTP API — works from anywhere
- 🖥️ Cross-platform HWID helpers built in
