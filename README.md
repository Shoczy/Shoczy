### Shoczy

license keys, HWID locking, user logins and gated downloads.

- SDKs: [lumenauth-sdks](https://github.com/Shoczy/lumenauth-sdks) (JavaScript, Python, Rust, C#)
- Working mostly with TypeScript, Next.js, Bun, Rust and Postgres.
